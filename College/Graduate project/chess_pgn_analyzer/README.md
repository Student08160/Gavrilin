\# Chess PGN Analyzer



\## Описание проекта



Веб-приложение для хранения и анализа шахматных партий формата PGN.  

Система поддерживает регистрацию пользователей, загрузку шахматных партий, просмотр партий на интерактивной доске, построение статистики и оценку позиции с помощью движка Stockfish.



\## Основные возможности



\- регистрация и авторизация пользователей;

\- личный кабинет;

\- загрузка PGN-файлов;

\- хранение партий по пользователям;

\- статистический анализ партий;

\- круговая и столбчатая диаграммы;

\- таблица популярных дебютов;

\- просмотр партии на шахматной доске;

\- переключение ходов вперёд и назад;

\- оценка позиции с помощью Stockfish;

\- сохранение истории анализа в базе данных.



\## Используемые технологии



\- Python

\- FastAPI

\- Jinja2

\- SQLite

\- SQLAlchemy

\- python-chess

\- Chart.js

\- HTML / CSS / JavaScript



\## Установка зависимостей



Перед запуском проекта необходимо установить зависимости:




pip install -r requirements.txt

Настройка Stockfish
Скачать Stockfish с официального сайта:
https://stockfishchess.org/download/

Выбрать версию для Windows 64-bit.

Распаковать архив.

Переименовать исполняемый файл движка в:

stockfish.exe
Поместить файл в папку:

engine/
Итоговый путь должен быть таким:

engine/stockfish.exe
Запуск проекта
Запуск приложения выполняется командой:

uvicorn main:app --reload
После запуска приложение будет доступно по адресу:

http://127.0.0.1:8000
Тестовые данные
Для демонстрации проекта можно использовать файл:

test_games.pgn
